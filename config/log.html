# Log 相关参数介绍

日志分为两种，一种是控制台产生的日志。一种是文件日志。其中控制台日志不可更改一律为第 3 命令行日志默认为 2 可以通过 Loglevel 修改

# Loglevel

日志输入等级分为 1-5 级分别如下

  1. 会记录所有的日志，比如一些调试日志、插件运行日志、所有的原始请求
  2. 会记录部分日志，包含所有的 POST 请求日志
  3. 只记录一些告警日志还有插件功能中开启的 dump 模块的日志
  4. 是记录错误日志
  5. 系统严重错误日志。基本是不可运行

# log_file

指定日志文件输出的位置
